Strength & Mobility for Barre Chords

Develop finger strength, clean fretting, seamless shape shifts, and rhythmic endurance.

1. Partial Barre Drill

Start with two-string mini-barre: finger 1 covers strings 1+2, then 1+3, up the neck.

// 2-string partial at fret 5
e|--5---5---5---5---|
B|--5---5---5---5---|
G|--x---x---x---x---|  // mute by touching with index knuckle
D|------------------|
A|------------------|
E|------------------|

// Move up one fret each measure
// 5→6→7→8 at ♩=60, 4 bars each

2. E-Shape Barre Moves

Full E-shape barre: root on the 6th string. Practice sliding up and down.

// 1st fret E-shape (F major) → 3rd fret (G) → 5th (A)
e|--1-------3-------5-------3---1---|
B|--1-------3-------5-------3---1---|
G|--2-------4-------6-------4---2---|
D|--3-------5-------7-------5---3---|
A|--3-------5-------7-------5---3---|
E|--1-------3-------5-------3---1---|

// Slide smoothly between each shape, 4 reps each direction

3. A-Shape Barre Moves

A-shape barre: root on the 5th string. Focus on clean bass note and barre stability.

// 2nd fret A-shape (B chord) → 4th fret (C#) → 5th fret (D)
e|--2-------4-------5-------4---2---|
B|--2-------4-------5-------4---2---|
G|--2-------4-------5-------4---2---|
D|--4-------6-------7-------6---4---|
A|--2-------4-------5-------4---2---|
E|--x-------x-------x-------x---x---|

// Move up the neck by 2 frets each cycle, 3 reps

4. Shape Shifting Patterns

Alternate between E-shape and A-shape on the same root for fluid transitions.

// Key of G, E-shape on 3 → A-shape on 10
// Cycle: E-shape → A-shape → E-shape
e|--3-------10------3-------|
B|--3-------10------3-------|
G|--4-------11------4-------|
D|--5-------12------5-------|
A|--5-------12------5-------|
E|--3-------10------3-------|

// Shift every two beats at ♩=70, 8 bars total

5. Rhythmic Barre Strums

Combine barre shapes with strumming patterns: quarter, eighths, syncopated.

// G major E-shape barre at 3rd fret
// Quarter note strum
| D   D   D   D |

// Eighth note strum
| D U D U D U D U |

// Syncopated (eighth + rest)
| D - U D - U D - |
| 1   & 2   & 3   & |

// Each pattern, 4 bars on shape, then switch to A-shape

6. Endurance Routine

Build forearm strength and stamina by long holds and repeat cycles.

// Hold G E-shape barre (3rd fret) for 30s
// Immediately switch to A-shape (10th fret), hold 30s
// Repeat 3× with 15s rest between

// Barre cycling at ♪=60
| E-shape | A-shape | E-shape | A-shape |
|    4     |    4    |    4     |    4    |  // bars

Practice Plan

  • Partial barré (section 1) – 5 min
  • E-shape slides (section 2) – 5 min
  • A-shape slides (section 3) – 5 min
  • Shifting patterns (section 4) – 5 min
  • Rhythmic strums (section 5) – 5 min
  • Endurance holds (section 6) – 5 min
