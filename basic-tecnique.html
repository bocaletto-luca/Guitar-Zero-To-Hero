1. Chromatic Warm-Up

Start slow at quarter = 60, get precise under the fingers, then gradually speed up.

// Across all six strings
e|---------------------------------0-1-2-3-2-1-0---------------------------------|
B|---------------------------0-1-2-----------------2-1-0---------------------------|
G|---------------------0-1-2-----------------------------2-1-0---------------------|
D|---------------0-1-2-------------------------------------------2-1-0---------------|
A|---------0-1-2-------------------------------------------------------2-1-0---------|
E|---0-1-2-------------------------------------------------------------------2-1-0---|

2. Single-Note Precision

Sustain and release cleanly. Work string by string.

// 4 beats per string
e|--0---0---0---0---|  
B|--1---1---1---1---|  
G|--2---2---2---2---|  
D|--3---3---3---3---|  
A|--2---2---2---2---|  
E|--0---0---0---0---|

3. Alternate Picking

Keep strict down/up movement. Start on one string, then two-string patterns.

// One-string pattern (e-string)
e|--0 D U 0 D U 0 D U 0 D U--|  // D=downstroke, U=upstroke

// Two-string pattern
e|--0---0---0---0---|
B|----1---1---1---1-|
     D U D U D U D U

4. String Muting Drill

5. Chord Transition Drill

Switch smoothly between Em → G → C → D at ♩=70.

| Em |  G  |  C  |  D  |  
e|--0-----3-----0-----2--|

G|--0-----0-----0-----2--|

A|--2-----2-----3-----x--|
E|--0-----3-----x-----x--|  ×4 cycles

1. Intro to Arpeggios

An arpeggio is a broken chord—notes played in sequence rather than simultaneously. Arpeggios build finger-independence and chord awareness.

2. Basic 4-Note Patterns

Start with simple major and minor open-chord shapes.

// C major
e|------0-------0-------0-------0---|
B|----1---1---1---1---1---1---1---1-|
G|--0-------0-------0-------0-------|
D|3-------3-------3-------3---------|
A|----------------------------------|
E|----------------------------------|

// G major
e|------3-------3-------3-------3---|
B|----0---0---0---0---0---0---0---0-|
G|--0-------0-------0-------0-------|
D|0-------0-------0-------0---------|
A|2-------2-------2-------2---------|
E|3-------3-------3-------3---------|

// A minor
e|------0-------0-------0-------0---|
B|----1---1---1---1---1---1---1---1-|
G|--2-------2-------2-------2-------|
D|2-------2-------2-------2---------|
A|0-------0-------0-------0---------|
E|----------------------------------|

3. Diatonic Sequences

Move across a single scale—here’s C major triad arpeggio up one octave.

// C major (root on 5th string)
e|-------------------------------------8--|
B|-----------------------------8--10------|
G|-----------------------9--12------------|
D|-----------------10-14------------------|
A|--3--7--10-7--3-------------------------|
E|----------------------------------------|

4. Arpeggio Combinations

Alternate between two shapes for musical flow.

// C → Am combo at ♪=80
C shape:
e|------0-----|
B|----1---1---|
G|--0-------0-|
D|2-----------|
A|3-----------|
Am shape:
e|------0-----|
B|----1---1---|
G|--2-------2-|
D|2-----------|
A|0-----------|

5. Practice Routine

  • Basic patterns (ex. #2) – 5 min
  • Diatonic sequence (ex. #3) – 5 min
  • Combos (ex. #4) – 5 min
  • Freestyle arpeggio – 5 min
